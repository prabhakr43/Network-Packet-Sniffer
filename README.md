
# Network Packet Sniffer & ARP Detector

This project is a Python-based security tool developed during my internship at SkillFied Mentor. It demonstrates how to monitor local network traffic and identify potential Man-in-the-Middle (MITM) attacks.

### Key Features
* **Live Sniffing:** Captures raw data packets from the network interface.
* **ARP Analysis:** Specifically monitors ARP responses (Opcode 2).
* **Security Alerts:** Identifies suspicious MAC-IP binding changes in real-time.

### Technical Stack
* **OS:** Kali Linux / macOS
* **Language:** Python 3.x
* **Library:** Scapy
